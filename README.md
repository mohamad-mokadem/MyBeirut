# MyBeirut

MyBeirut is a municipality service management platform developed for CSC490 (Software Engineering). The platform was designed to help citizens submit and manage municipality-related requests through a centralized web application.

## Features

- User authentication using Firebase
- Municipality request submission and tracking
- Admin and user dashboard interfaces
- Database integration and request management
- Responsive web interface

## Technologies Used

- PHP
- MySQL
- Firebase
- HTML/CSS/JavaScript

## Contribution

Worked as a full-stack developer in a team of 5. Mohammad Daoud and I were mainly responsible for backend development, database management, Firebase authentication, and backend-related functionalities.

## Purpose

The project was developed as part of CSC490 (Software Engineering) to simulate the development of a real-world municipality management platform.
